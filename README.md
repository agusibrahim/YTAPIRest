# YouTube ~Private~ API
> Access youtube private API via REST

## Feature
- search suggestion
- search video/music
- playlist
- detail video
- detail stream url
- translate

No login required, just deploy and access it!

## How to run? SIMPLE!
Using docker make your life easier

`docker-compose up -d`

Access via `http://localhost:3061`
